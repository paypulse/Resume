어린이집·유치원 교사 및 학부모용 맞춤형 유아 놀이 큐레이션 AI

아이 개개인의 성향과 발달 수준을 반영한 맞춤형 교육 정보 제공 플랫폼

개발 배경

아이 관련 정보가 SNS와 커뮤니티에 흩어져 있어 필요한 정보를 제때 찾기 어렵다는 문제를 발견했습니다.

"내 아이를 위한 사교육은 존재하지 않는다"

엄마이자 개발자의 입장에서 아이 맞춤형 정보를 제공하는 서비스를 만들고자 본 프로젝트를 시작했습니다.

주요 기능

  • ✔ 맞춤형 유아 놀이 큐레이션
  • ✔ AI 기반 태그 매칭
  • ✔ 통합 교육 정보 제공
  • ✔ 공공/민간 데이터 연동
  • ✔ 학부모 후기 커뮤니티

기술 스택

Backend

Spring Boot, Kotlin, QueryDSL, Spring Batch

Frontend

Vue.js, Vite

AI / Data

Drools, LLM API, Crawling

Infra

Docker, Jenkins, AWS, Redis, MQTT
